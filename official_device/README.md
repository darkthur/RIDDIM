# CerberusOS Project Official Device Repository

  This folder is containing official devices information. The files is generated when you build CerberusOS saved in cbi file(CerberusOS Build Info)
  
## Make device official
  Here is tep to make official
  - Make a new repository in your personal github with name
    "cerberus_bRAND_CODENAME" e.g "cerberus_xiaomi_ferrari"
  - Put following files in the repository
    * cbi file
    * device tree
    * vendor tree
    * kernel tree
  - Send the repository link to device.cerberus@gmail.com with title and put the ROM Link
    "Official device submit"
